Edit B4P on Windows

B4P programs are standard text files in ANSI or UNICODE formaT. B4P will recognize the character format automatically. The Editor App, also and formerly known as Notepad is a pre-installed editor on Windows systems. Use the Microsoft file browser (Explorer) to visit directory C:\Users\Your user name\Documents\Programs\B4P. Choose the B4P file (for example: Hello World.b4p and click on the right mouse button and choose Edit. It is typically the 2nd menu entry and could appear in a different language, depending on the language settings of the system.


You can also use alternative text editors as long they do not embed any text formattings such as different fonts, colors, underscoring, etc. Visual Studio Code is also a popular editor for Windows and other platforms.
