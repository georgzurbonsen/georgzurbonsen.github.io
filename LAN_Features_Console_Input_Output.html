Console Output

The consoles make use of the UNICODE character set. Some older or specialized consoles may not support all characters found in the UNIOCDE space and may output placeholders or substitute characters. Some modern consoles support the extended multilingual planes which include emojies.

Console Input

Consoles under Windows have only limited capabilities to input non-ANSI characters. Under Windows, foreign characters are accepted, but the EURO-symbol is not.
Hint: Consider using character entities to specify special characters which you do cannot find on your keyboard.
