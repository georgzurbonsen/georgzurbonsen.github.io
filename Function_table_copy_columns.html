Function Names

table copy columns, table copy columns selected rows

Description

This function copies contents from selected columns to another set of selected columns.

table copy columns selected rows
copies the contents only in rows meeting the conditional expression in the 2nd function parameter.

Call as: procedure

Restrictions

Indirect parameter passing is disabled

Parameter count: 3, 4

Parameters

No.TypeDescription
1.
input		 literal Name of existing table

Opt. 2.
code		 expression
:literal		 Expression to select rows

Applicable to function table copy columns selected rows only. Specify the conditions or rules to selecte the rows. See expressions to select rows.
Partial table specification is supported here to reference table columns easily. Example: [Score]>=5.
Attention: Put the condition into parentheses if you use the more flexible operators '=' and '<>' instead of '==' and '!='.

2 / 3
input		 table columns Existing columns

Specify header names and/or column numbers which are considered as part of the headers

2 / 3
input		 table columns New columns

Column numbers or header names of destination columns.
Missing header names will be created on the right hand side of the table.
Column numbers outside existing range creates columns with blank header names.

Exceptions

Table not found
Header name not found

Examples

 include( Support Library );

 table initialize ( table,
     { { First Name, Last Name, Given Name, Birth Year },
       { Abel,       Adams,     A.,         1990 },
       { Betty,      Beavers,   B.,         1995 },
       { Charlie,    Collins,   C.,         2000 },
       { Daniela,    Douglas,   D.,         2005 } } );

 table copy columns selected rows( table, [Birth Year]<1998, {First Name, Last Name}, { Given Name, Family Name } );
 // The specified columns in the first two rows will be copied

 table list( table );
