Entities

Tables in supported HTML / MHTML, as well as softquoted literals (referenced in program code with single quotation marks) support entity references. Examples:

Entity Character Explanation
&euro; € Euro sign (case sensitive, i.e. &Euro; will not be converted)
&#8364; € ", represented with decimal UNICODE number
&#x20AC; € ", represented with hexadecimal UNICODE number
&#X20ac; € Same. The letters X and of hexadecimal digits are not case sensitive
