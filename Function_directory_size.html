Function Names

directory size

Description

Returns the total number of bytes used by all files specified in the directory, including all their subdirectories. Hidden and inaccessible files will not be counted.

Call as: function

Restrictions

Indirect parameter passing is disabled

Parameter count: 1

Parameters

No.TypeDescription
Opt. 1
input		 literal path and directory name

Specify the directory name (subdirectory in working directory) or path name. Do not use wildcards.

Return value

TypeDescription
numeral Total size

Size of all files identified, summed, unit is bytes.

Exceptions

Specified directory not found or not existing

Examples

     // Function 'literal' adds thousand separators to make numbers readable
     echo( literal( directory size( "C:\Program Files\Beyond4P" ), "#,000", local ) );
     echo( literal( directory size( "outputs" ), "#,000", local ) );

Output

10'147'907
2'408'216
