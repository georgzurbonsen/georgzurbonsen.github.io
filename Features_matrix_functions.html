Functions

Beyond4P provides a library of powerful matrix functions. While vectors are modeled as 1-dimensional parameter sets containing numerals, matrices are modeled as parameter sets (rows) containing parameter sets containing __numerals (columns). Example: {{ 1, 2, 3 }, { 4, 5, 6 }} is a matrix consisting of 2 rows and 3 columns:

| 1 2 3 |
| 4 5 6 |
