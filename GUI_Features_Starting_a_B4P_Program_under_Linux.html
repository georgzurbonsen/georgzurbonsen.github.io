Starting a B4P Program under Linux - Hello World

Two possibilites exist to start B4P programs:

  • Start B4P Programs as Executables under Linux, e.g. by typing ./"hello world.b4p" .
    Good to know: Wildcards are allowed as long the file does not become ambiguous, e.g. ./hello* .
  • Linux Start B4P to run B4P programs, e.g. b4p "hello world.b4p" .

Quotation marks are needed on files if they contain spaces.
