Function Names

table copy table columns, table copy table columns selected rows

Description

This function makes a copy of the table with all selected columns.
The function table copy table columns selected rows copies only the selected rows instead of all rows. If the new table is already existing, then it will be initialized.

Call as: procedure

Restrictions

Indirect parameter passing is disabled

Parameter count

2

Parameters

No.TypeDescription
1.
input		 literal Name of existing table

2.
input		 literal Name of new table

3.
input		 table columns Header names and/or column numbers

See table columns as function parameters for general ruling for this parameter.
Specific rules apply for this function:

  • Any number of header names and column numbers may be specified

4.
code		 expression
:literal		 Expression to select rows to copy

Applicable for table copy table columns selected rows only: The selected rows will be copied into the new table.
Specify the conditions or rules to select the rows. See expressions to select rows.
Partial table specification is supported here to reference table columns easily. Example: [Score]>=5.
Attention: If you are using comparison operators = and <> (instead of == and !=) which allow more than 1 operand on the righ hand side , then make sure you put the entire expression into parentheses. Otherwise the operators will hijack subsequent function parameters as part of their operands.

Examples

  table initialize( t, {{ Name, Age, Hobby },
      { Ann, 45, Skiing }, { Dan, 35, Skating }, { Eva, 40, Bee keeping }, { Nic, 10, Reading } } );

  echo("Original table:");
  table list( t );

  echo("Copy Hobby and Name:");
  table copy table columns ( t, u, {Hobby, Name} );
  table list( u );

  echo("Copy Name and Age of young people:");
  table copy table columns selected rows ( t, v, {Name, Age}, [Age]<40 );
  table list( v );

Output

Original table:

    1 : Ann  | 45  | Skiing     
    2 : Dan  | 35  | Skating    
    3 : Eva  | 40  | Bee keeping
    4 : Nic  | 10  | Reading    

Copy Hobby and Name:
    0 : Hobby       | Name
    1 : Skiing      | Ann 
    2 : Skating     | Dan 
    3 : Bee keeping | Eva 
    4 : Reading     | Nic 

Copy Name and Age of young people:

    1 : Dan  | 35 
    2 : Nic  | 10
