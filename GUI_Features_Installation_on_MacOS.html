Introduction

The following pages provide step-by-step instructions for downloading and installing the B4P applications for MacOS. In the example documented here, the UBUNTU distribution has been used.

System requirements:

  • Computer: Macintosh laptop, iMac, or Mac Mini running the latest version of MacOS
  • Processor: 64 bit
  • Memory: Minimum 2 GB; More memory is recommended especially when working with large data
  • Disk space: 20 MB
  • Internet connectivity: Required to access documentation, but not during normal program execution
  • Administrator privileges: You must have privileges to install software on your computer (e.g. allowed to run commands with 'sudo' in front)

Software requirements: Download, install, and verify that a code editor, terminal application, and web browser is available and working. Recommended editor, terminal application, and web browsers are as follows:

  • Code editor:
    • Visual Studio Code: https://code.visualstudio.com/docs/setup/mac
  • Terminal application:
    • iTerm: https://iterm2.com
  • Web browser:
    • Firefox: https://www.mozilla.orgfirefox
    • Chrome: https://www.google.com/chrome
    • Safari: https://support.apple.com/downloads/safari

Terminal application requirements Open the Terminal application (iTerm) and install the following terminal commands as follows:

  • Add the terminal package manager 'homebrew'.
    • Instructions: https://brew.sh
  • Add the terminal command 'wget' using homebrew.
    • Instructions: https://www.maketecheasier.com/install-wget-mac
  • Add the terminal command 'code' to open any file or folder from the terminal
    • Instructions: https://code.visualstudio.com/docs/setup/mac
