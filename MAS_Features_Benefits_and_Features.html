Introduction

B4P, with its full name Beyond Former Expectations, consist of two main building blocks:

  • The B4P Language
  • The B4P Analytics Engine

The B4P Analytics Engine is a powerful and yet light-weight interpreter which executes the B4P language reliably and with high performance and is proven to run on Windows, LINUX and MacOS platforms with both 64-bit Intel/AMD and ARM processors.
