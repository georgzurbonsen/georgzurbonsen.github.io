Overview

B4P supports six basic data types for doing calculations in expressions and storing data in variables. However, tables store all data in text form, including numbers. Numerals written to tables are converted into numbers in text form, and numbers read are converted into numerals.

Strings and numerals are also categorized in two subtypes each which are desribed in the following sections.

  • Numerals
    • Numerals with associated text representations
    • Plain numerals
  • Strings
    • Softquoted strings
    • Quoted strings
  • Dates
    • This type can contain dates, times, combination of both, and blank dates.
  • Booleans
  • Sets
    • They contain multiple elements of values of any data type, including nested sets. Empty sets (0 elements) are supported.
  • Voids
