Introduction

The following pages provide step-by-step instructions for downloading and installing the B4P applications for MacOS.

Computer Requirements

  • Computer: Macintosh laptop, iMac, or Mac Mini running the latest version of MacOS
  • Processor: 64 bit
  • Memory: Minimum 2 GB (More memory is recommended especially when working with large data)
  • Disk space: 20 MB
  • Internet connectivity: Required to access documentation (but required for program use)
  • Administrator privileges: You must have privileges to install software on your computer (e.g. allowed to run commands with 'sudo' in front)

Software Requirements

Verify that a code editor, terminal application, and web browser are available and working. Recommended editor, terminal application, and web browsers are as follows:

  • Code editor: Visual Studio Code. Download here:
    • https://code.visualstudio.com/docs/setup/mac

  • Terminal application: iTerm Download here:
    • https://iterm2.com

  • Web browser: Firefox, Chrome, or Safari. Download here:
    • https://www.mozilla.orgfirefox
    • https://www.google.com/chrome
    • https://support.apple.com/downloads/safari



Terminal Application Requirements

Open the Terminal application (iTerm) and install the following terminal commands:

  • 'brew'. This provides a capability to install packages
    • Instructions: https://brew.sh
  • 'wget'. This provides capability to download any files from the internet.
    • Instructions: https://www.maketecheasier.com/install-wget-mac
  • 'code'. This provides capability to open any text file with Visual Studio Code
    • Instructions: https://code.visualstudio.com/docs/setup/mac
