Edit B4P on Windows

B4P programs are text files which are either in ANSI or UNICODE format. B4P will recognize the character format automatically. Use the Microsoft file browser (Explorer) to visit directory C:\Users\Your user name\Documents\Programs\B4P. Choose the B4P file (for example: Hello World.b4p and click on the right mouse button and choose Edit. It is typically the 2nd menu entry and could appear in a different language, depending on the language settings of the system.


Edit B4P with Windows Commands

Change to the directory where the B4P program file is located, then type notepad "hello world.b4p". Quotation marks are only required if the file name contains spaces. You may use alternative text editor such as Visual Studio Code, too.
