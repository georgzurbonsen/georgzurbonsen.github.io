Function Names

add color

Library

Style Library

Description

This function adds a further color to the color palette.

Call as: procedure

Restrictions

Don't forget to include 'Style Library' at the top of your program

Parameter count

5

Parameters

No.TypeDescription
1.
input		 literal New color name

Specify a new color name which does not yet exist in the color table

2.
input		 literal Category 2

Apply a categorization, e.g. an existing one like "gray scale" or a new one of choice. This value will noted down as "Cat 2" in the color table. The 1st category is assigned automatically with "user defined".

3.
input		 numeral Red intensity

Valid values: 0..255

4.
input		 numeral Green intensity

Valid values: 0..255

5.
input		 numeral Blue intensity

Valid values: 0..255

Examples

  include( Style Library );

  add color( crème, natural shades, 245,238, 222 );

  table initialize( t, { { Item, Color }, { Jacket, crème }, { Trousers, gray 12 }, { scarf, teal } } );

  table process( t, table style cells( t, Color, row(), single, fill color, [Color] ) );

  translate style attributes for html(t);
  table save( t, "Images/Custom_Colors.html", HTML );
  translate style attributes for excel(t);
  table save( t, "Images/Custom_Colors.xls", EXCEL );
  echo("Done. Open the file Custom_Colors.html with a web browser.");

Output

Done. Open the file Custom_Colors.html with a web browser.
Try it yourself: Open REF_Function_add_color.b4p in B4P_Examples.zip. Decompress before use.
