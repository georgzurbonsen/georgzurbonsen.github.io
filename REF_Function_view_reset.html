Function Names

view reset

Library

Support Library

Description

This function deletes all temporary HTML files beginning with "_b4p_view... .html". These files are create by the view() command when viewing B4P tables using web browsers. It is advised to use this function in interactive mode only.

Call as: procedure

Restrictions

Under normal conditions, the 'Support Library' is loaded automatically, so no 'include(...)' call is needed.

Parameter count

0
