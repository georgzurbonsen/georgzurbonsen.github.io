Function Names

file existing

Description

These functions check if the specified file is existing

Call as: function

Restrictions

Indirect parameter passing is disabled

Parameter count: 1

Parameters

No.TypeDescription
1
input		 literal path and file name

If no path is specified, then the working directory is checked.

Return value

TypeDescription
boolean Result

true if specified name is existing as a file.
false if not existing or existing as a directory

Examples

     echo( file existing( "C:\Program Files" ) );
     echo( file existing( "C:\Program Files\Beyond4p.exe" ) );

Output

false
false
