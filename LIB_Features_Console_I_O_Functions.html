Introduction

The following functions deal with basic input/output using the console.

  • Text input/output covers basic functions like input() and echo().
  • Console special effects adds clear screen, cursor control and some color to the console.
  • Inspection and debugging contain functions which are helpful for interactive use, e.g. to inspect variables.



Procedures and Functions Provided:


Text to standard output:
    echo
    print
    print line

Text composition:
    compose
    compose line

Text from standard input:
    input

Direct character input from console:
    input quick
    input quick ignore case
    getch
    getch ignore case

clear the screen:
    cls

cursor position, move cursor:
    cursor

font color, foreground color, screen color:
    background color
    text color

Change the prompt used in interactive mode:
    prompt
    inspect

List variables including their members and data types:
    list global variables
    list local variables
    list system variables
    list variables
