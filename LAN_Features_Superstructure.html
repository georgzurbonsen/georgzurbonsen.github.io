Introduction

In B4P, the Superstructure defines the different kinds of statements and how they are put together and how they are integrated in blocks in order to write B4P programs, and, as advanced users, your own function libraries.

Blocks and Statements



You will recognize that this language structure has overall similarities to C/C++ and Java.

  • All B4P programs consist of 1 or more Statements or Blocks in any order.
  • All Blocks are encapsulated with braces { ... } .
  • Nested blocks are allowed.
  • All statements are delimited with semicolons ; .
