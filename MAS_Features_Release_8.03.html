Bugfixes

* Loading JSON files into B4P tables: Loading refused if JSON begins with array and not with object. Has been fixed. * table lookup() family: Stabilization (eliminated a potential bug) * Release 8.02 induced an error in the function table distribute() which prevented doing the work. Has been fixed.

New Functions

* table digest() function family: New variant of lookup and integration functionality * table lookup with rules() (also ... digest, ... expand, ... integrate) introduced. * Additional help functions, e.g. help keywords, help functions, etc. Recommended for interactive use

Improvements in Existing Functions

* table lookup smart() (also ... digest, ... expand, ... integrate): Added option to specify B4P expression for more flexible comparisons

New Language Features (e.g. syntax and semantics)

None in this release.

Performance Improvements

* Did a spring-season clean-up of the existing function family table lookup() / table integrate() / table expand() and shoud show minor performance improvements.
