Introduction

Long options are command line options beginning wtih two consecutive minus signs (--) followed by an option name. In addition, option values may be assigned to that name if immediately followed by an equal sign and further values. For example b4p test.b4p --silent --use_value=3 specifies two long options. The long options are passed as command line parameters to the B4P program and can be used for individual purposes. In contrast to short command line options, it does not interprete the long options as they are solely provided for use by B4P programs.

See the B4P function description of get long options() where some code examples are shown.
