Introduction

Beyond4P provides a wide variety of control flow functions which include conditional branches and loops. In contrast to other programming languages, trivial features such as the if() statement is actually a Beyond4P function which influences execution the next statement or block.

In case you miss the goto statement, there is a good reason for this. The highly structured Beyond4P interpreter would have faced risky challenge if that statement would have been impleemented.
