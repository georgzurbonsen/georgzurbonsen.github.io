Introduction

B4P has the unique capability to translate characters received with the functions getch() or input quick() into dedicated UNICODE characters situated in the Private Use Area (PUA). Every key pressed, e.g. a function or arrow key, will return a single character.

Note: The operating system may capture some of the keys for their own use, e.g. for application or task switching. Try the keys out for possible operating system functionality before using them in the B4P program.

B4P provides dedicated character entities for all these characters.
