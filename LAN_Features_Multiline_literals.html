Introduction

Similar to Python and other modern programming languages, B4P supports multline strings, for both softquoted literals and quoted literals. Softquoted multiline strings begin with three single quotation marks ''' with contents across multiple lines and concluding with '''. For quoted multiline strings, use """ delimiters instead. The three quotation mark symbols must be written without blanks in-between.

The string contents are treated similarly as regular string constants, for example character entities like &szlig; being resolved to the sharp S in German: ß if inside 3 consecutive ''' symbols, but not inside """ symbols. The length of multi-line strings is presently limited to 200,000, where (foreign) non-ANSI characters count 2..4 bytes depending on their code (UTF8 symbols). Longer strings will be invalidated and result in error messages.

Inside the multi-line literals, 1 and 2 consecutive quotation markes are treated as regular text. 3 consecutive quotation marks are also treated as text if that specific one is not used as the opening delimiter. B4P comments inside multi-line strings are also treated as part of the string contents and not as code comments, e.g. to exclude some contents.

Multiline string contents can start immediately after the three quotation symbols or on the next row. Same applies for the end: The multi-line string can be ended on the same line with three quotation marks, or immediately on the following row. If you want to include new line symbols at the beginning and/or at the end of the multi-line string, then include additional blank rows.

Attention: If the multi-line string is not properly delimited with the same triple quotation marks, then an error message will occur at the end of the B4P program file because B4P assumed all contents further down as part of the long string.

special offer[] = '''

    4 pounds of bananas cost &euro; 2.90.
    """ This is today's special bargain """

''';

echo("Attention!", special offer[], "As long supplies last?" );
Attention!
    4 pounds of bananas cost € 2.90.
    """ This is today's special bargain """
As long supplies last?

code example[] = """
    /* Comments are part of the multi-line string */

    a[] = 4;
    b[] = sqrt(a[]) + 1; // Square root calculation
    echo( "    sqrt(4)+1 equals ", b[] );
""";

echo(  code example[] );
echo( new line, "Code output:" );
exec( :code example[] );
