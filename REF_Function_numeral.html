Function Names

numeral

Description

This function converts a variable to a plain numeral.

Call as: function

Restrictions

Indirect parameter passing is disabled

Parameter count: 1

Parameters

No.TypeDescription
1
input		 valid types Value

Literals containing numbers (no text before numbers) are converted properly. Scientific notation is supported.
Numerals are passed through. If existing, the literal context of the number will be discarded
Booleans are converted to 0 and 1
Parameter sets are not accepted
Dates are converted to a number in YYYYMMDD. Time will be added as fraction of 1, e.g. 0.5 for 12:00 noon.

Return value

TypeDescription
numeral number

Contains resulting number. No literal context of the number is preserved or included.

Examples

     echo( numeral( '1.2E+03' ) );
     echo( numeral( true ) );
     echo( numeral( 0123 ) );
     echo( numeral( date ( "05. Nov 2019 12:00" ) ) );

Output

1200
1
123
20191105.5
