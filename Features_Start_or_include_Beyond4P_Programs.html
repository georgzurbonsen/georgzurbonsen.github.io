Introduction

Beyond4P provides start() to start other Beyond4P programs. Execution of this calling program continues when the Beyond4P program has ended. The include() behaves similarly like start(), but memorizes the loaded program including the defined functions. This features is useful to define and include user specific function libraries which can be re-used in different Beyond4P programs.
