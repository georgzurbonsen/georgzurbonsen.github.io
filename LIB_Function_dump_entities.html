Function Names

dump entities

Description

This function generates a table which contains all availbale character entities supported by B4P. Character entities are descriptive representations of characters which are not available on the keyboard. They begin with an ampersand (&), followed by a descriptive string or character code and conclude with a semicolon (;).

The table contains the official entities supported by web browsers as well as the proprietary entities used by B4P only which are special characters in the UNICODE Private Use Area (PUA) for special keyboard characters (e.g. navigation keys) and console output features (e.g. text colors and attributes).

The table contains following info:

Entity Name Name of character entitiy, beginning with ampersand (&) symbol and ending with semicolon (;)
Decimal UNICODE value in decimal representation
Hex UNICODE value in hexadecimal representation: U+0000 - U+FFFF and U+010000 - U+10FFFF
Output Character (as long they are visible and non-proprietary characters)
Official 'Yes'or 'No': Yes means that the character entity is also supported by web browsers.
Remarks Distinguishes between ANSI, UNICODE and B4P-proprietary characters

Call as: procedure

Restrictions

Indirect parameter passing is disabled

Parameter count

1

Parameters

No.TypeDescription
1
input		 string Name of table

A table will be created using this name.

Examples


    dump entities( table );
    table list ( table );
