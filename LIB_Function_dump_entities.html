Function Names

dump entities

Description

This function creates a B4P table which contains all availbale HTML like entities, for example &ge; for ≥ (greater than or equal). The table also lists entities which are unofficial, i.e. not common in HTML code, and characters belonging to the UNICODE Private Use Area (PUA) which B4P uses for console features (colors, text attributes, navigation and function key inputs).

Note: Numeric entities, e.g &027; or &x1b; are not listed here.


The table contains following info:

Entity Name Entity Name, always beginning with ampersand (&) symbol and ending with semicolon (;)
Decimal UNICODE value in decimal format
Hex UNICODE value in hexadecimal format. Format: U+0000 - U+FFFF, U+010000 - U+10FFFF
Char Actual Character (as long it is a visible character and not in the PUA range).
Official Contains Yes or No. Yes means that the entity is also supported by web browsers.
Remarks Indicates whether the character relates to the B4P specific

Call as: procedure

Restrictions

Indirect parameter passing is disabled

Parameter count

1

Parameters

No.TypeDescription
1
input		 literal Name of table

A table will be created using this name.

Examples


    dump entities( table );
    table list ( table );
