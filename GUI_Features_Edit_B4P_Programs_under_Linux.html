Edit B4P under Linux

B4P programs are standard text files in ANSI or UNICODE formaT. B4P will recognize the character format automatically. LINUX provides a wide selection of editors, such as vi, emacs, gedit and Visual Studio Code. They are all fine as long the do not embed any text formattings such as different fonts, colors, etc.
