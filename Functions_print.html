Function Names

print, print line, echo, compose, compose line

Synopsis

print (...)
print line (...)
echo (...)
compose (...)
compose line (...)

Description

print outputs contents to the screen without next line and provides them as return value, too.
print line: works like print, but adds new line at the end.
echo: works like print line, but indirect parameter passing is suppressed, so if the 1st parameter is a parameter set, it will be treated as such
compose: outputs contents to return value only. No screen output
compose line: works like compose, but adds new line at the end.
The values provided through the parameters will be converted to literals before outputting them

Call as: procedure or function

Restrictions

Function 'echo': No indirect parameter passing

OS differences

None

Parameter count: Min. 0

Parameters

No.TypeDescription
all
input		 all types Values

The values provided will be part of the output.

Return value

TypeDescription
literal Output

Output string generated resp. printed on screen

Exceptions

none

Examples

     print('Hello '); // Without line break
     echo(World); // With line break
     echo("My name is ", system info[user name] );
     echo( { This, is, a, parameter, set } ); // Output a parameter set
     
     // While echo(...) suppresses indirect parameter passing, the print... functions
     // treat the parameter set provided as 1st parameter as actual list of function parameters.
     print line( { This, is, a, parameter, set } );
