Introduction

Tables are the key objects on which this programming language has been developed for, namely to process tables of any size effectively and efficiently with minimum amount of code to be written and delivering maximum possible execution performance. All tables are 2-dimensional data structures which hold all information as strings (text), including numeric values, boolean values, dates and parameter sets.
