Overview

Release 9.02, updated 2022-03-XX, "William Henry Hoover"
Inventor of the vacuum cleaner, and a huge time-saving benefit for everyone.

Bugfixes

  • Specifying "." (with double quotation marks) has not enforced searching for header name called ".". This has been fixed
  • Fuction input() - Bugfix (issues with blank prompts resolved)

New Functions

None in this release

Improvements in Existing Functions

  • for all table rows: Can accept as few as 1 parameter.
  • for all table selected rows: Can accept as few as 2 parameters.

New Language Features (e.g. syntax and semantics)

None in this release

Other new Features

None in this release

Clean-ups and Streamlining

  • The type literal consist of two subtypes (softquoted literal and quoted literal) and no longer three.
  • Various clean-ups done in B4P docs

Performance Improvements

  • Clean-up in the B4P engine results in minor performance improvements.

Depreciated and retired features

Softquoted literals: Header names and likes in a single string and separate with commas will no longer be separated into individual parameters as it created more confustion and processing time than providing benefits. Put multiple parameters into a parameter set instead.

The function type detailed has been renamed to subtype.
