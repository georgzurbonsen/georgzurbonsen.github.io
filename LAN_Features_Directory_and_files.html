Introduction

Special care has been taken with accessing directories and files as well as specifying directory and file names in order to achieve an optimum cross-platform portability, able to run on Windows, UNIX/Linux and MacOS platforms. They cover issues as little as directory separator symbols in path names, using the right line break character sequences, chosen character sets and country-specific differences in files considered highly standardized and generic which they are not.
