Introduction

The B4P language is designed to solve problems with as few clearly understandable statements as possible, allowing to keep the number of statements, variables and loops as low as possible.

Overall Syntax

The overall syntax of B4P is closely related to C/C++ wich provides

  • Statements are separated with semicolons ;
  • Statements may take multiple lines of text
  • { } for defining language blocks and other syntax structures
  • Physical code indentation (like in Python) to describe syntax structures is not applicable in B4P.
  • Same comment symbols are used as in C/C++
  • B4P is case sensitive.
