Overview

Release 12.00, issued 2025-06-01, "Johann C. F. Gauss"
1777-1855, mathematican, geodetist, astronomer and physicist.

Bugfixes

  • Some error messages involving sets provided in variables referred to the wrong location in the code, making debugging more challenging. Has been fixed.

New Functions

  • table column numbers() added to return positive table column numbers are providing (mix of) table header names and column numbers

Improvements in Existing Functions

New Language Features (e.g. syntax and semantics)

  • Function parameters table columns now support column ranges, provided as softquoted strings, e.g. '..'.

Other new Features

Clean-ups and Streamlining

  • No license file required anymore when installing B4P

Performance Improvements

  • No changes

Depreciated and retired features

  • for all parameters() is replaced by for all() (Still existing, but no longer recommended to use)
