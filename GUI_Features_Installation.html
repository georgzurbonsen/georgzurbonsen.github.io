Introduction

The following pages describe the installation procdure for the following target systems:

  • Windows (at least Windows 7)
  • Linux
  • MacOS

Since the installation script has also been implemented in B4P, the installation procedure appears a bit differently as compared to installing other software, however you will be experiencing a very fast installation procedure. All installations are machine-based, i.e. different users can access B4P on the same machine.

Attention: Local administrator rights are needed to install B4P, similar to intalling any other software on your machine.
