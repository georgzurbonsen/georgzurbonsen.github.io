Function Names

table rename column headers, table rename existing headers

Description

This function renames specified table headers. Whereas table rename column headers asserts exceptions if headers are not found, the table rename existing headers will simply ignore the headers not found.

Call as: procedure or function

Restrictions

Indirect parameter passing is disabled

Parameter count: 2

Parameters

No.TypeDescription
1
input		 literal Name of existing table

2
input		 table columns Existing header name(s) or column number(s)

Specify any number of table header names and/or column numbers.

3
input		 parameter set or literal New header name(s)

List the new header names corresponding to the existing ones specified in the previous function parameter.
Use literal to to specify a single table, or a parameter set or softquoted literal (single quotation marks) with names separated by comma to specify multiple columns.

Return value

TypeDescription
numeral Number of renamings done

Exceptions

Table not found
Header name not found
number of existing and new headers is inconsistent

Examples

 include( Support Library );

 table initialize ( table,
 { { Last Name, First Name, Phone No, Age }, { Tanner, Jim, "1-212-555-1212", 28 } } );


 table rename existing headers( table, { Last Name, 1st Name, First Name }, ' Family Name, Family Name, Given Name ' );
//    table rename existing headers( table, { Last Name, 1st Name, First Name }, { Family Name, Family Name, Given Name } );
//    Both formulations are valid (see 3rd function parameter: Either softquoted literal with comma separated itesms or parameters)
 table list ( table );

Output

Family Name  Given Name  Phone No        Age  
Tanner       Jim         1-212-555-1212  28
