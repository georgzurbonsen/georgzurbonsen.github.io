Function Names

blinking

Description

This function activates / deactivates blinking on the output console. If no parameter is provided, then blinking will toggle. Deactivation takes place when B4P ends.

Call as: procedure

Restrictions

Indirect parameter passing is disabled

OS differences

Some consoles, terminals and terminal emulators may not support this feature.

Parameter count

0,1

Parameters

No.TypeDescription
Opt. 1
input		 boolean Activated

true activates it, false deactivates it.

Default value: Toggling: Previous setting will be reversed
