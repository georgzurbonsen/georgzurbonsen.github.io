Function Names

table copy table

Description

This function makes a copy of the table with all contents. If the new table is already existing, then it will be initialized.

Call as: procedure

Restrictions

Indirect parameter passing is disabled

Parameter count

2

Parameters

No.TypeDescription
1.
input		 literal Name of existing table

2.
input		 literal Name of new table

Examples

  table initialize( t, {{ Name, Age }, { Ann, 45 }, { Dan, 35 }} );
  table copy table ( t, u );
  table list ( u );

Output

    0 : Name | Age
    1 : Ann  | 45 
    2 : Dan  | 35
