Overview

Release 9.01 tbd, updated 2022-01-10, "The Golden Cut"
Golden Cut - or Golden Ratio - is the same as the ratio of their sum to the larger of the two quantities. Common in mathematics, modern arts and architecture.

Bugfixes

  • table view(): Minor correctios

New Functions

  • file browser() provides an interactive and feature-rich file browsers to select files
  • existing here(), existing here and valid(): Returns true if the variable is defined as a local variable or provided through a function parameter. This prevents unfavorable influences from equally named global variables.

Improvements in Existing Functions

  • width() and text row() ensure that formatting characters such as changing colors are not counted as full width character.

New Language Features (e.g. syntax and semantics)

  • Various new HTML like entities have been introduced for UNICODE characters in the PUA (Private Use Area) in order to do text formatting and colors on the console, as well as recognizing special keyboard keys such as function and navigation keys.
