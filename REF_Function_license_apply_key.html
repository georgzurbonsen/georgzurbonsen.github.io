Function Names

license apply key

Description

Normally, you do not need to call this function manually as it is taken care by the installation script. The license provided will be checked and, if valid, stored in the registry so Beyond4P will refer to this one at start-up. If the call is successful (no exceptions), then the license key is memorized and stays active until further notice.

Call as: procedure

Restrictions

Indirect parameter passing is disabled.

Parameter count

3

Parameters

No.TypeDescription
1
input		 literal Key

24-byte key in hexadecimal format

2
input		 literal License holder name

License holder name. This value is case sensitive and must correspond with the license key.

3
input		 boolean For machine or user installation

true: License key stored in appropriate directory for machine installations;
false: License key stored in appropriate directory for current user installations,
