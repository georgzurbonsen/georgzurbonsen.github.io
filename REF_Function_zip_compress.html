Function Names

zip compress

Library

Zip Library

Description

Compresses a file (or directory with all the contents) into a ZIP file

Call as: procedure

Restrictions

Under normal conditions, the 'Zip Library' is loaded automatically, so no 'include(...)' call is needed.

Parameter count

1-3

Parameters

No.TypeDescription
1.
input		 literal Source path and file names

Choice of file or directory name. If directory name is chosen, then the whole directory with all contents will be compressed. Wildcards e.g. ('*.*, 'pictures\*.jpg') are allowed. In this case, the 2nd function parameter (Archive file name) is also required to provide a valid archive file name.

Opt. 2.
input		 literal Archive file name

Required if 1st parameter contains wildcard symbols to compress multiple original files.

Opt. 3.
input		 literal Archive type

Valid values: zip and 7z. The latter does only apply to Windows compilation where 7Zip is used.
