Function Names

table style table

Library

Style Library

Description

This function applies style and formatting to a whole table or sheet

Call as: procedure

Restrictions

Don't forget to include 'Style Library' at the top of your program

Parameter count

4, 6, 8, etc.

Parameters

No.TypeDescription
1.
input		 literal Name fo existing table

2.
input		 literal Formatting Coverage

See formatting coverages. Valid values are:

  • sheet: Covers entire sheet, even outside the table
  • table: Covers table, but sheet outside the table is not affected
  • body: Covers table except header row

3, 5, ...
input		 literal Generic Attribute Name

See generic attribute names for available names.

4, 6, ...
input		 literal Generic Attribute Value

See generic attribute values for valid values. Different rules apply for the different generic attribute names.

Examples


  table load( t, "Examples\Cities.csv" );

  table style table( t, sheet, column width, 24, wrap text, true );
  table style table( t, body, text color, navy );

  table save( t, "Images/Style_Function_table_style_sheet.html", HTML );
  table save( t, "Images/Style_Function_table_style_sheet.xls", EXCEL );
  table save excel file( t, Contents, "Images/Style_Function_table_style_sheet.xlsx" );
  echo("Done.");

Output

    Saved table(s) to EXCEL file 'Images/Style_Function_table_style_sheet.xlsx' containing 1 sheets.

Done.
Try it yourself: Open LIB_Function_table_style_table.b4p in B4P_Examples.zip. Decompress before use.
