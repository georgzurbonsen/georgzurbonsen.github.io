Function Names

help

Library

Support Library

Description

This function provides an overview of the most important features provided in interactive mode such as accessing help and support information, and how B4P statements and expressions can be executed directly as commands.

This help feature provides user interaction allowing to browse through different pages.

Call as: procedure

Restrictions

Under normal conditions, the 'Support Library' is loaded automatically, so no 'include(...)' call is needed.

Parameter count

0

'help' entered in interactive mode
