Introduction

The B4P function fuamily list variables() and inspect() aer suitable to inspect the contents of the variables and their members if these variables contain array and/or structure members.

Command Description
list variables
list variables(a)
list variables('a*') 		All variables inclding their members and sub-members will be listed, including data types and applicable protections.
Limit the listing varialbes named 'a'
Limit the listing to all variables beginning with 'a' (made use of wildcard symbols).
list system variables Same as above, but system variables only
list global variables Same as above, but global variables only
list local variables Same as above, but local variables only
inspect( variable name[] )
inspect( a[] )
inspect( a[b] ) 		Inspect the contents of the variable including all defined array and/or structure members if existing.
Example for variable 'a[]'
Example for member variable 'a[b]'
table list ( my table ) List the table. See table list() for details where you can specify additional parameterse to specify the starting and ending rows and columns and also use the 'briefly' option to list the first and last n lines only. The table will be shown on the console without need of user interaction and is also useful as a program statement to print out table contents.
view ( my table ) This function provides visualzation with user interaction (e.g. navigation) using
- The console output
- Excel
- A web browser of choice
- A web browser of choice with data tables.
If no further parameter is provided, then the last browser option chosen will be used.
view ( my table, here ) Visualization on the console, with some features to navigate
view ( my table, excel ) Opens the file with Excel
view ( my table, chrome ) Opens the file with a browser, for example Google Chrome (other browers available, see view() for details
view ( my table, chrome datatables ) Opens the files with a browser and making use of data tables, a function provided by SpryMedia Ltd. under the MIT license.
= a[] Use execute B4P expressions interactively to access values in individual variables.

Examples

In the follwoing example, some variables are created and then inspected interactively.

>> a[] = 3
>> a[kid] = 4
>> a[nic] = 5
>> inspect(a[])
a[]                     3  ("3")                   (numeral,full access)
kid                     4  ("4")                   (numeral,full access)
nic                     5  ("5")                   (numeral,full access)

>> inspect(a[kid])
a[kid]                  4  ("4")                   (numeral,full access)

>> =a[]
3 [numeral]
>> list variables( a )

Variable List Level = 0  Regional Level = 0    System variables
_______________________________________________________________________________

Variable List Level = 1  Regional Level = 1    Global variables
_______________________________________________________________________________
a                       3  ("3")                   (numeral,full access)
  kid                   4  ("4")                   (numeral,full access)
  nic                   5  ("5")                   (numeral,full access)
