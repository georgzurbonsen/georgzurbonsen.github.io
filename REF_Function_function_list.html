Function Names

function list, function explain

Description

These functions lists provide information about the available functions

Call as: procedure

Restrictions

Indirect parameter passing is disabled

Parameter count

1

Parameters

No.TypeDescription
1
input		 literal Function name search string

The search string narrows down the name of the functions to be included in the listing. Wildcard symbols are supported here.

Examples

    define procedure( mode, {{ meaningful text, literal }} )
    {
        echo("Test");
    }

    echo("Functions listed:");
    function list( "mo*" ); // Lists 3 functions: mod, mode (defined here) and month.
    echo;
    echo("Functions explained:");
    function explain( "mo*" );

Output

Functions listed:
mod
mode
month

Functions explained:
mod  (Function)
  2 parameters
    1: divisor, type: numeral
    2: dividend, type: numeral

mode  (Procedure)
  1 parameters
    1: meaningful text, type: literal

month  (Function)
  1 parameters
    1: date, type: to date (Values which are dates or can be converted to dates)
