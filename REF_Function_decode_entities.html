Function Names

decode entities

Description

This functiond decodes entities (like € ac; €) into actual characters. The common entity names used in HTML are supported.

Note: Entities inside softquoted literals specified in the program code will be converted automatically. See examples below.

Call as: function

Restrictions

Indirect parameter passing is disabled

Parameter count

1

Parameters

No.TypeDescription
1
input		 literal input string

This string contains entity-encoded special characters

Return value

TypeDescription
literal String with decoded characters

Converted string with decoded characters

Exceptions

Invalid character code (outside 0 - 1114111 range)

Examples

     echo( decode entities( "4.50 &euro; / 4.50 &#8364; / 4.50 &#x20ac; / 4.50 &#X20AC;" ) );
     echo( "4.50 &euro;" ); // Quoted literal are not decoded.
     echo( '4.50 &euro;' ); // Softquoted literal is already decoded implicitly.
     echo( '4.50 &eur' + 'o;' ); // This one is not because full entity is not in one softquoted literal value

Output

4.50 € / 4.50 € / 4.50 € / 4.50 €
4.50 &euro;
4.50 €
4.50 &euro;

Try it yourself: Open REF_Function_decode_entities00.b4p in B4P_Examples.zip. Decompress before use.
