Function Names

inspect

Description

This function checks inspects a variable and displays all contents including those of all memeber variables. The types are displayed as well as protection settings. Regarding types: 'literal' means softquoted literal and "literal" means quoted literal.

Hint: This function is suitable for interactive and debugging use in order to inspect variables conveniently.

Call as: procedure

Restrictions

Indirect parameter passing is disabled

Parameter count

1

Parameters

No.TypeDescription
1
code		 variable
:literal		 Variable to inspect

Examples

    simple[] = 0123.00;
    array( words[], { 1, Me, you, true } );
    words[1] = { one, 2, { 2.5, two point six, 2.8 } };
    structure( sizes[], { whale, dog, ant }, {huge, small, tiny } );
    structure( sizes[dog], { jack russel, poodle, setter }, { smaller, medium, bigger } );

    inspect(simple[]);
    inspect(words []);
    inspect(sizes []);
