Function Names

identify

Synopsis

identify( variable name );

Description

This function identifies the form of the variable, i.e.
- A simple variable without any further members
- A structure with at least one member inside
- An array with at least one member inside
- A zero-member array
- Or the variable is not existing (no exception asserted in this case)
Zero members arrays are close relative to simple variables with the only difference that provisions have been made to contain variable members. Zero member array occur when deleting all members from the structure or array. Redimensionig (e.g. using redim() arrays to 0 elements result in simple variables. Zero members arrarys also apply when loading a JSON file containing empty arrays. B4P is able to clearly distinguish between null and empty arrays.

Call as: function

Restrictions

No indirect parameter passing

Parameter count: 1

Parameters

No.TypeDescription
1
code		 variable
:literal		 variable name

Any variable name, regardless if it is a base variable or a member

Examples

     a[] = "It's simple";
     array( b[], { Item 1, Item 2 } );
     structure( c[], { red, blue }, { rot, blau } );

     echo( "a[]: ", identify(a[]), "  b[]: ", identify(b[]), "  c[]: ", identify(c[]), "  d[]: ", identify(d[]) );

     echo( "b[0]: ", identify(b[0]), "  b[2]: ", identify(b[2]) );

     echo( "c[red]: ", identify(c[red]), "  c[green]: ", identify(c[green]), "  c[red,yellow]: ", identify(c[red,yellow]) );
     delete( c[red], c[blue]);
     echo( "c[] without members: ", identify(c[]) );

     delete( b[0] );
     delete( b[0] );
     echo( "b[] without members: ", identify(b[]) );

     echo( "Try with json" );
     json to variable( v[], '{ "k" : 12, "l" : { "l1" : 23, "l2" : 34 }, "m" : [ 45, 56 ], "n" : [] }' );

     echo( "Member k: ", identify(v[k]) );
     echo( "Member l: ", identify(v[l]) );
     echo( "Member m: ", identify(v[m]) );
     echo( "Member n: ", identify(v[n]) );

Output

a[]: simple  b[]: array  c[]: structure  d[]: not found
b[0]: simple  b[2]: not found
c[red]: simple  c[green]: not found  c[red,yellow]: not found
c[] without members: zero members
b[] without members: zero members
Try with json
Member k: simple
Member l: structure
Member m: array
Member n: zero members
