Function Names

encode entities

Description

This function encodes special characters into entities. Following characters are encoded:

  • > - Greater Than -> &gt;
  • < - Less than -> &lt;
  • " - Double quotation marks -> &quot;
  • ' - Single quotation marks -> &#39;
  • & - Ampersand symbol -> &amp;
  • All control characters (e.g. new line, tab, backspace, escape) -> &#nnn;
  • Control character code 0 (NUL) will be ignored.
  • All non-ANSI characters (e.g. é) -> &#nnn;
  • 2nd and further consecutive spaces are converted to &#32;

Call as: function

Restrictions

Indirect parameter passing is disabled

1-2

Examples

      echo( encode entities( '4.50 €  <TABLE> "..." </DIV>' ) );
      echo( encode entities( Line 1 + new line + Line 2, false ) ); // use &#10;
      echo( encode entities( Line 1 + new line + Line 2, true ) ); // use <BR/>
