Function Names

docs

Library

Support Library

Description

This function opens the PDF file stored on your machine which is part of the B4P installation. Just type 'docs', a PDF viewer opens with the documentation and you can continue working.

Call as: procedure

Restrictions

Under normal conditions, the 'Support Library' is loaded automatically, so no 'include(...)' call is needed.

Parameter count

0
