Function Names

clean numeral, clean if numeral

Description

This function converts a literal into numeral, including eliminating redundant contents in commercial number representations such as currency symbols before or after themand other form of units. Thousand separators will be removed and decimal commas will be converted to decimal points. Minus signs both before and after the numerals will be detected. Numbers in parentheses, e.g. (100.00) = -100 are considered negative numbers. Scientific notation (e.g. 1.2E+03 ) is supported here. Numerals passed as parameters into thes function will be passed through. If the 1st parameter contains no number (e.g. letters, blanks) , then clean numeral returns 0 and clean if numeral returns the original literal.

Call as: function

Restrictions

Indirect parameter passing is disabled

Parameter count

1-2

Parameters

No.TypeDescription
1
input		 numeral or literal Value

If literal and containing digits: Conversion to numeral.
If literal and containing no digits or literal is blank: Conversion to 0 (clean numeral) or literal returned (function clean if numeral) If numeral: Will be passed through.

2
input		 literal Decimal symbol

Use single character symbol, e.g. '.' or ','.
Specify 'local' to use currently applicable locale settings. Examples: It chooses '.' for countries like USA, United Kingdom and Switzerland, and ',' for Germany and Austria.

Default value: . (point)

Return value

TypeDescription
numeral
literal		 number

Contains resulting number. Literals may be returned by the function clean if numeral if the input value contains no digits.

Examples

     echo( clean numeral ( "EUR 1'234.50" ), " and ", clean if numeral ( "EUR 1'234.50" ) ); // Both 1234.5
     echo( clean numeral ( "2,500 € " ),     " and ", clean numeral ( "2,500 €", ",") ); //  2500 and 2.5
     echo( clean numeral ( "2,500- € " ) ); // -2500
     echo( clean numeral ( "-   2,500" ),    ", ",    clean numeral ( "(2,500)" ) ); //  Both -2500
     echo( clean numeral ( "[2,500]" ),      " and ", clean numeral ( "-[2,500]" ) ); // 2500 (brackets alone do not negate)
     echo( clean numeral ( "(EUR 1,000)" ),  " and ", clean numeral ( "(EUR) 1,000") ); // Negation if number INSIDE parentheses
     echo( clean numeral ( "INR 30 40 500" ) ); //  3040500
     echo( clean numeral ( ".123" ),         " and ", clean numeral ( ".123", "," )); // returns 0.123 and 123.
     echo( clean numeral ( "" ),             " and '",clean if numeral( "" ), "'" ); // returns 0 and "";
     echo( clean numeral ( "abc" ),          " and ", clean if numeral ( "abc" ) ); //  returns 0 and abc

     echo( new line, "Scientific Notations:");
     echo( clean numeral ( 1.2E3 ), " and  ", clean numeral ( '1.2E+03' ) ); //  return 1200
     echo( clean numeral ( "1 2 E - 0 3" ) ); // returns 0.012
