Introduction

No dedicated formatting template is foreseen for formatting parameter sets. For advanced features to convert parameter sets to literals with opening, closing and separator strings, use the join() function instead.

Examples:
 echo( literal( { 1, true, 'true', {b1, b2, { b2a, b2b }, b3 } } ) );
