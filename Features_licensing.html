Licensing Scheme

Without possessing a license, Beyond4P will start in a demo mode and exploit the full performance. The demo mode is useful for evaluations, demo, education and doing small projects. However, following restrictions apply:

Table size restrictions when saving Tables can only be saved if the table contains not more than 300 rows and not more than 40 columns.
Number of savings During a running Beyond4P session, at most 5 tables can be saved. The number of tables (and not files) also applies when using the procedure table save multiple to save two or more tables in one file.

Table size restrictions do not apply while processing tables internally, i.e. you are able to load large amounts of data and do data processing and analysis, but need to condense your output accordingly inorder to meet the size limitations.

Individual licenses for one machine, a specific domain or site licenses will be provided on request.
