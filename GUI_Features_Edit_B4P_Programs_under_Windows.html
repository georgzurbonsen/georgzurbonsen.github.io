Edit B4P under Windows

B4P programs are standard text files in ANSI or UNICODE format. B4P will recognize the character format automatically. The Editor App, also and formerly known as Notepad is a pre-installed editor on Windows systems. Use the Microsoft file browser (Explorer) to visit directory C:\Users\Your user name\B4P. Choose the B4P file (for example: Hello World.b4p and click on the right mouse button and choose Open with and then choose the editor. Depending on your language settings, the menus may appear in a different language.

You can also use alternative text editors as long they do not embed any text formattings such as different fonts, colors, underscoring, etc. Visual Studio Code is also a popular editor for Windows and other platforms.
