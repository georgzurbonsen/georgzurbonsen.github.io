Introduction to Date Functions

Beyond4P provides some powerful functions to process date and time information (particularly the variable type date) as well as stopwatch functions. The sections are:

  • Date functions
  • Time functions
  • Stopwatch functions

The following important date functions are found in the section Type Conversion Functions and connvert data of other types, e.g. literals, to dates.:

  • date()
  • pure date()
  • date time()
  • time()
  • pure time()
