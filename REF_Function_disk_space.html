Function Names

disk space, disk space free

Description

Returns the total resp. free space of the specified drive. It also works for network drives.

Call as: function

Restrictions

Indirect parameter passing is disabled

Parameter count: 1

Parameters

No.TypeDescription
Opt. 1
input		 literal drive or path name

Specify a disk drive (e.g. C:\) or a path name (/usr/local, or \\server_name\path_name\...)

Return value

TypeDescription
numeral Size information

Disk space / free disk space in bytes

Exceptions

Invalid path name or nonexisting drive

Examples

     echo( literal( disk space( "C:\" ), "#,000", local ) );
     echo( literal( disk space free( "C:\" ), "#,000", local ) );

Output

234'483'609'600
90'142'511'104
