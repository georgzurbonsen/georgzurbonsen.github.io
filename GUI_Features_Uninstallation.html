Introduction

Use the same installation software to de-install B4P. All program an library images will be cleared. Under Windwos, all registry configurations will be retracted, for example assigning the ".b4p" file type to the B4P icon and removing B4P in the path environment variable.

Your personal B4P program folder created by the installer will not be removed. You can always re-install B4P after a deinstallation has been made
